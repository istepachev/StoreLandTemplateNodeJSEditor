-Шаблонизатор для html файлов.
Шаблонизатор над шаблонизатором :) Возможность выносить блоки кода в отдельный html файл с применением переменных.
-Плагин Babel для JavaScript файлов
Возможность писать в современном синтаксисе (const,let, ()=>{}) с транспиляцией в (var, function(){})
-Плагин SCSS для Css файлов
Возможность использовать преимущества scss (переменные, разделение на отдельные файлы, миксины и тп.)
-Плагин Prettier для автоформатирования кода (JS, Css)
Ставит нужные отступы/переносы и точки с запятой. Приводит к единому стилю код.
-Возможность работы с несколькими версиями шаблона из одной папки
Переделана файловая структура, переключение и слияние изменений между версиями за счёт git.
